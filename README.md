PIPO -THE MULTILINGUAL VOICE ASSISTANT

An intelligent virtual assistant (IVA) or intelligent personal assistant (IPA) is a software agent that will perform tasks or deliver services as a result of a user's privately supported inquiries or instructions. A virtual assistant that can be accessed through web chat is at times referred to as a "chatbot." Online chat programs may sometimes be utilized only for entertainment. There are virtual assistants that can understand verbal commands and respond using synthesized voices. Users may use voice commands to organize other mundane tasks such as email, to-do lists, and calendars besides inquiring from their assistants, home automation device control, and media playback control. One of the finest uses of artificial intelligence is the virtual personal assistant (VPA), which provides an alternative method for humans to assign work to machines. For developing a virtual personal assistant (VPA) and employing it in software applications, there are some approaches and principles employed. For allowing the users to interact with virtual assistants, speech recognition systems—also referred to as automatic speech recognition, or ASR—are very important.
